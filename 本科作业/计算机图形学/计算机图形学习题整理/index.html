# 习题 6.15

# 题目

试用 Sutherland-Hodgman 算法对如图 6-41 所示的多边形进行裁剪，要求画出每次裁剪对应的图形，并标明输入和输出的顶点。

# 解答




  1. 用左⬅️边界裁剪
    输入 ABCDE
    输出 12ABCD
  2. 用下⬇️边界裁剪
    输入 12ABCD
    输出 1234B56D
  3. 用右➡️边界裁剪
    输入 1234B56D
    输出 1234B7D
  4. 用上⬆️边界裁剪
    输入 1234B7D
    输出 1234B789
